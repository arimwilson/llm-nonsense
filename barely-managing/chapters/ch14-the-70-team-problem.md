# Chapter 14: The 70-Team Problem

Draft status: scaffold complete.

## Chapter intent
Diagnose and fix throughput problems in large engineering organizations.

## Core anecdote
Multiple teams hit local goals while portfolio delivery slips globally.

## Key sections
- Dependency debt mapping.
- Org design as throughput architecture.
- Governance that reduces handoff drag.

## Concrete managerial decision
Reorganize around value streams instead of legacy component boundaries.

## Manager action checklist
- Map critical-path dependencies.
- Reduce handoff layers.
- Set escalation ownership.
